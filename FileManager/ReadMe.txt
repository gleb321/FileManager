Справка по командам:
Команда help:
Выводит в консоль информацию о доступных командах из файла "help.txt", так что файл "help.txt" лучше не трогать)

Команда vdisk:
Выводит в консоль список дисков компьютера.

Команда sdisk:
Команда для выбора диска. Рекомендуются при запуске программы первым делом выбрать диск, тк без выбранного диска
работают только следующие команды: help, vdisk, sdisk, stop.

Команда vfile:
Выводит в консоль имена всех файлов, находящихся в текущей директории.

Команда vdir:
Выводит в консоль имена всех директорий, находящихся в текущей директории.

Команда mtdir:
Команда для перехода в директорию, доступную из текущей директории.
Gyde по использованию:
Допустим сейчас мы находимся в директории C:\Users\Username, из которой доступны директории dir1, dir2 и тд
(для просмотра доступных директорий воспользуйтесь vdir). Для того, чтобы перейти в dir1, вводим команду 
mtdir, после чего нажимае ENTER и вводим dir1. Так же если вы знаете, что в дирекотрии dir1 содержится директория
subdir1, то в нее тоже можно попасть, введя dir1\subdir1 (dir1/subdir1 в MacOs).

Команда cd:
Возвращение в корневую директорию диска

Команда pwd:
Выводит в консоль путь до текущей директории

Команда stop:
Завершает работу файлового менеджера

*********************************************************************************************************************

Далее следуют команды для работы с файлами. Для работы с конкретным файлом, вы должны находиться в той же директории, 
что и файл. Gyde по кодировкам:
utf8 - UTF-8
utf32 - UTF-32
unicode - Unicode
acsi - ASCII
Также учтите, что большинство текстовых редакторов не работают с кодировками, отличными от UTF8, поэтому для ввода текста
в файл рекомендуется использовать команду wtfe, для просмотра содержимого файла - cwfe.

Команда cwf:
Выводит содержимого выбранного файла в консоль в кодировке UTF-8

Команда cwfe:
Выводит содержимого выбранного файла в консоль в выбранной пользователем кодировке

Команда copyf:
Cоздает копию выбранного пользователем файла в той же директории.
Копия файла name.txt будет называться name-copy.txt.
Повторное копирование файла name.txt приведет к ошибки, тк файл с именем name-copy.txt уже существует

Команда fmtdir:
Перемещает выбранный пользователем файл в выбранную пользователем директорию.
При вводе директории, нужно указать абсолютный путь до нее.
Допустим у нас есть файл, путь до которого выглядит так: C:\Users\Username\file.txt, и мы хотим переместить его в директорию
Username, тогда абсолютный путь до Username выглядит так: C:\Users\Username.

Команда delf:
Удаляет выбранный пользователем файл.

Команда crf:
Cоздаёт текстовый файл в кодировке UTF-8.
При вводе имени файла не нужно указывать расширение, файл автоматически будет создан с расширением .txt.

Команда crfe:
Cоздаёт текстовый файл в выбранной пользователем кодировке
При вводе имени файла не нужно указывать расширение, файл автоматически будет создан с расширением .txt.

Команда ctf:
Команда записывает текст из двух тектовых файлов в кодировке UTF-8 в третий файл в кодировке UTF-8.
Допустим у нас есть два файла с именами first.txt и second.txt, тогда при объединении их в один файл, будет 
создан файл first_second.txt

Команда ctfe:
Команда записывает текст из двух тектовых файлов в третий файл в выбранной пользователем кодировке
При вводе сначала нужно указать имена двух файлов, которые вы хотите объединить, после чего нужно ввести три кодировки
(кодировку первого файла, кодировку второго файла, кодировку итогового файла)
Название итогового файла строится аналогично ctf

Команда wtfe:
Записывает текст из файла input.txt в выбранный пользователем файл, в выбранной пользователем кодировке
Путь до файла из папки проекта выглядит так: FileManager\bin\Debug\net5.0\input.txt
