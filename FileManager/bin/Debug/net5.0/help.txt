help - вывод списка доступных команд
vdisk - список дисков компьютера
sdisk - выбор диска
vfile - просмотр списка файлов в данной директории
vdir - просмотр списка директорий в данной директории
mtdir - переход в другую директорию
cd - возвращение в корневую директорию
pwd - просмотр текущей директории
cwf - вывод текста из файла в консоль в кодировке UTF-8
cwfe - вывод текста из файла в выбранной пользователем кодировке
copyf - создает копию файла в той же директории
fmtdir - перемещения файла в выбранную пользователем директорию
delf - удаление файла
crf - создаёт текстовый файл в кодировке UTF-8
crfe - создаёт текстовый файл в выбранной пользователем кодировке
ctf - записывает содержимое двух файлов в третий
ctfe - записывает содержимое двух файлов в третий в выбранной пользователем кодировке UTF-8
wtfe - записывает текст из файла input.txt в выбранный пользователем файл, в выбранной пользователем кодировке
stop - выход из файлового менеджера
